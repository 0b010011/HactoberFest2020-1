# Blinker
A Programming Language

## Features
- Performs all the binary operations.
- Variable creations.
- Error Handling.

## Create a variable:
```text
> 3 + 3       // 6
> 3 ^ 2       // 9
> 3 - 1       // 2
> 4 / 2       // 2
> 8 * 2       // 16
> 1 + (2 * 3)  // 7
> 10 / 0      // Division By Zero Error
> vr variable_name = variable_value
```
